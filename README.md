# 🚀 우주선 탄막 게임 (Space Shooting Game)

## 📖 프로젝트 소개
파이썬(Python)과 Pygame을 활용하여 제작한 2D 우주선 탄막 게임입니다.  
플레이어는 우주선을 조작하여 장애물과 적을 피하고, 자동으로 발사되는 공격을 통해 적을 제거하며 점수를 획득합니다.

---

## 🎮 주요 기능

- ⌨️ 방향키로 우주선 이동 (상/하/좌/우)
- 💥 자동 공격 시스템 (탄환 자동 발사)
- 👾 적 생성 및 충돌 판정
- 🧱 장애물(벽) 충돌 시 게임 오버
- ⏸️ 일시정지 / 재시작 기능
- 📖 게임방법 팝업 UI 제공
- 🏆 랭킹 시스템 (이름 입력 및 점수 저장)

---

## ⚙️ 게임 규칙

1. 우주선을 이동하여 장애물과 적을 피합니다.
2. 자동으로 발사되는 공격으로 적을 제거합니다.
3. 적을 제거하면 점수가 증가합니다.
4. 벽이나 적과 충돌하면 게임 오버입니다.
5. 게임 종료 후 이름을 입력하여 랭킹에 기록할 수 있습니다.
